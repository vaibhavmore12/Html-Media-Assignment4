HTML, CSS, JavaScript
HTML (Hypertext Markup Language), CSS (Cascading Style Sheets), and JavaScript form the core front-end technologies. HTML structures web content, CSS styles it, and JavaScript adds interactivity.
Node.js
Node.js is a JavaScript runtime that allows server-side development. It's used for building scalable and high-performance server applications.
Express.js
Express.js is a web application framework for Node.js. It simplifies the creation of robust and scalable APIs and web applications.
React
React is a JavaScript library for building user interfaces. It's commonly used for creating interactive, single-page web applications.
Angular
Angular is a front-end framework that helps build dynamic web applications with a structured architecture and built-in tools for various tasks.
Vue.js
Vue.js is a progressive JavaScript framework for building user interfaces. It's known for its simplicity and flexibility in creating web applications.
SQL/NoSQL Databases
SQL databases like MySQL and PostgreSQL, and NoSQL databases like MongoDB and Redis, are used for storing and managing data in web applications.
RESTful/API
RESTful APIs (Representational State Transfer) are used for communication between client and server in web applications, allowing data to be exchanged in a standardized way.
Git/GitHub
Git is a version control system, and GitHub is a web-based platform for hosting and collaborating on code repositories. They are essential for version management and teamwork.
